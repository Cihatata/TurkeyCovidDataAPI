# TurkeyCovidData
The source of this information is the [ministry of health.](https://covid19.saglik.gov.tr)

## Installation

```bash
git clone https://github.com/Cihatata/TurkeyCovidDataAPI.git 
cd TurkeyCovidDataAPI
npm install  
node app.js 
```
